# 🚀 TotalBot 빠른 시작 가이드

## 1단계: 서버 실행 (5분)

```bash
# 터미널 열기
cd "/Users/evans/Documents/테스트/server"

# 의존성 설치
npm install

# 서버 실행
npm start
```

✅ 성공 메시지가 나타나면 서버 준비 완료!
```
🚀 TotalBot 서버 시작됨!
   포트: 5000
```

---

## 2단계: 크롬 확장 설치 (2분)

1. **크롬 브라우저 열기**

2. **주소창에 입력:**
   ```
   chrome://extensions
   ```

3. **개발자 모드 활성화**
   - 우측 상단의 "개발자 모드" 토글 ON

4. **확장 프로그램 로드**
   - "압축해제된 확장 프로그램을 로드합니다" 클릭
   - 폴더 선택: `/Users/evans/Documents/테스트/chrome-extension`

5. **설치 확인**
   - 툴바에 TotalBot 아이콘이 나타남 (아이콘 없어도 작동)

---

## 3단계: 로그인 (1분)

1. **크롬 툴바에서 TotalBot 아이콘 클릭**

2. **로그인 정보 입력:**
   - 아이디: `admin`
   - 비밀번호: `admin123`

3. **로그인 버튼 클릭**

✅ 메인 화면이 나타나면 준비 완료!

---

## 4단계: 상품 수집 테스트 (2분)

1. **쿠팡 또는 알리익스프레스 상품 페이지 방문**
   - 예: https://www.coupang.com/vp/products/...
   - 예: https://www.aliexpress.com/item/...

2. **TotalBot 아이콘 클릭**

3. **"현재 페이지 수집" 버튼 클릭**

4. **수집 완료 확인**
   - 상품명, 가격, 이미지 개수가 표시됨

5. **"수집된 상품 보기" 클릭**
   - 엑셀 파일이 자동 다운로드됨

✅ 엑셀 파일이 다운로드되면 성공!

---

## 💡 주요 기능 사용법

### 📦 상품 수집
1. 쇼핑몰 상품 페이지 방문
2. TotalBot 아이콘 → "현재 페이지 수집"
3. "수집된 상품 보기"로 엑셀 다운로드

### 📋 발주 처리
1. "발주 처리" 탭 이동
2. "발주 파일 업로드" → 엑셀 선택
3. "발주 처리 시작" 클릭
4. 결과 확인

### 💰 정산
1. "정산" 탭 이동
2. "정산 파일 업로드" → 쿠팡/루트로지스 데이터 업로드
3. "정산 계산하기" 클릭
4. 정산서 다운로드

### ⚙️ 설정
1. "설정" 탭 이동
2. 쿠팡 아이디/비밀번호 입력
3. 사업자 번호 입력
4. "설정 저장" 클릭

## 🐛 문제 해결

### 서버가 시작되지 않음
```bash
# Node.js 설치 확인
node --version

# npm 설치 확인
npm --version

# Node.js 미설치 시:
brew install node
```

### 크롬 확장이 보이지 않음
- `chrome://extensions`에서 확장 프로그램 활성화 확인
- 오류 메시지가 있는지 확인
- 아이콘 파일이 없어도 작동함 (아이콘만 안 보임)

### 로그인 실패
- 서버가 실행 중인지 확인 (`http://localhost:5000/health` 접속)
- 아이디/비밀번호 확인: `admin` / `admin123`

### 상품 수집이 안 됨
- 지원되는 사이트인지 확인 (쿠팡/알리익스프레스/1688)
- 상품 페이지에서만 작동 (검색 결과 페이지는 미지원)
- F12 → Console에서 오류 확인
